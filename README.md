# Satellite Telemetry Anomaly Detection

This project focuses on detecting anomalies in satellite telemetry data using machine learning and data analysis techniques. It processes telemetry signals, identifies abnormal patterns, and improves monitoring reliability for space communication systems.

## Features
- Telemetry data preprocessing
- Anomaly detection models
- Data visualization
- Performance comparison
- Hybrid model training

## Technologies Used
- Python
- Machine Learning
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Structure
- preprocess.py
- train_model.py
- train_hybrid_models.py
- model_visualization.py
- model_comparison.py
- pipeline.py

## Applications
- Space communication monitoring
- Satellite health analysis
- Real-time anomaly detection
